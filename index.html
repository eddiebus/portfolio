Metta Mech

Final year video game group project. Working in a team of 9 members to develop our own video game in Unreal Engine 5 and showcased at the Derby University's Game Expo 2023 for the public and Industry guests. Metta Mech is a 2 player game of mech robots fending off against an invading alien horde. My contributions to the team were coding: UI, Gameplay and Graphic effects into the game using C++ and blueprints.

My contribution to this project was to develop UI systems and support in Gameplay and Graphics of the game. This includes:

  • Post-Process Shader Graphics
  • UI Creating With UMG
  • UI Navigation Support using C++
  • Pear-revewing Gameplayer code and bug fixing

Battle Of Gloo

Part of the Game Makter Toolkit Jam 2024. Joined a team of 5 to create a day in this 4 day jam. Our engine of choice was Unreal Engine(5.4). For this Jam we set to match the "Built to Scale" theme by making our battle game of gloos that batter and grow to become the strongest.

My contributions for this project was general programming across gameplay, ui and enemies. Prefered creating ai as the gloo don't follow the unreal engine character movement so they needed their own ai task to move along navigation paths.

RoboPunch

My 1st game "Robot Punch" made in Unreal Engine 5. A game of surviving and punching through endless wave of robots. This game was part of AI learning in University. I used random number generation in combination of Unreal Engine's behaviour tree system to create unpredicatble enemies that attack differently.

Unity Visual Audio
